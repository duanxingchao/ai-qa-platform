# Scripts 脚本目录

## 📁 目录结构

- `deployment/` - 部署相关脚本
- `development/` - 开发辅助脚本  
- `testing/` - 测试脚本
- `maintenance/` - 维护脚本

## 🚀 常用脚本

### 部署脚本
- `deployment/start_backend.sh` - 启动后端服务
- `deployment/start_frontend.sh` - 启动前端服务
- `deployment/start_mock_services.sh` - 启动Mock服务

### 维护脚本
- `maintenance/stop_scheduler_and_cleanup.py` - 停止调度器并清理数据
